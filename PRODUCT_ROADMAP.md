## ðŸ” Credit Bureau Integration Details

### UK Credit Reference Agencies

#### 1. Experian Connect API
- **Pricing:** Contact for quote (typically Â£0.50-Â£2 per credit check)
- **Features:** Credit score, full report, monitoring
- **Integration:** REST API, OAuth 2.0
- **Turnaround:** 2-3 months for approval + integration
- **Docs:** https://www.experian.co.uk/business/products/consumer-information/

#### 2. Equifax API
- **Pricing:** Usage-based, Â£1-Â£3 per check
- **Features:** Credit score, identity verification
- **Integration:** SOAP/REST API
- **Docs:** https://www.equifax.co.uk/

#### 3. TransUnion (Callcredit)
- **Pricing:** Custom pricing
- **Features:** Credit data, fraud detection
- **Integration:** REST API
- **Docs:** https://www.transunion.co.uk/

#### 4. ClearScore (Partnership Option)
- **Pricing:** Potential revenue share instead of API costs
- **Features:** Free credit scores for users, monetize via offers
- **Model:** White-label their service
- **Pros:** No cost per check, consumer trust
- **Cons:** Less control, revenue sharing

#### 5. Credit Kudos (Open Banking Alternative)
- **Pricing:** Contact for quote
- **Features:** Creditworthiness via Open Banking (no hard check)
- **Integration:** REST API, already aligned with TrueLayer
- **Pros:** Uses data you already have access to
- **Docs:** https://www.creditkudos.com/

### Recommended Approach:
**Start with Credit Kudos** for soft credit assessments using Open Banking data (already integrated with TrueLayer), then add **ClearScore partnership** for traditional credit scores without per-check costs.
